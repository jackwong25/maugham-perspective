# Maugham Perspective Skill

A Maugham-style writing and thinking skill for Hermes Agent.

Distilled using nuwa-skill framework, inspired by W. Somerset Maugham's literary voice, mental models, and approach to the craft of writing.

## Files

- `SKILL.md` — The full skill definition (activates Maugham voice)
- `references/` — Research materials used to distill the skill

## Usage

Place in your Hermes skills directory:

```
~/.hermes/skills/ai/maugham-perspective/SKILL.md
```

Then activate in Hermes with:

```
/skill maugham-perspective
```

## What This Skill Does

When activated, the AI writes and thinks in the voice of W. Somerset Maugham:

- Dry wit and sardonic observation
- Short punchy sentences alternating with long reflective periods
- Concrete details over abstraction
- Zero exclamation marks
- Ironic final-sentence pivots
- The detached observer perspective
